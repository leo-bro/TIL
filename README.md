## 🔥 주요 학습 내용
- **Git 초기화 및 설정**
  - 레포지토리 초기화 (`git init`) 및 사용자 정보 설정 (`git config`).
- **Git 기초 명령어**
  - 파일 추가 (`git add`), 커밋 (`git commit`), 상태 확인 (`git status`).
- **브랜치 관리**
  - 브랜치 생성, 전환, 병합 (`git branch`, `git checkout`, `git merge`).
- **고급 기능**
  - 리베이스, 충돌 해결, Stash 활용 (`git rebase`, `git stash`).
